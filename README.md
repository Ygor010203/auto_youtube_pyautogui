Automação simples em Python usando PyAutoGUI e Pyperclip para abrir o Google Chrome, acessar o YouTube automaticamente e pesquisar por um termo pré-definido, simulando as ações de teclado e mouse do usuário.
**Objetivo:** abrir o Chrome, entrar no YouTube e pesquisar uma música automaticamente.  

**Bibliotecas usadas:**  
- pyautogui  
- pyperclip  

**Como rodar:**  
1. Instale as dependências: `pip install pyautogui pyperclip`  
2. Execute o script com `python nome_do_arquivo.py`.
